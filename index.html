Reimplementing Apple's Paravirt GPU Compiler Plugin
The virtual iPhone uses a paravirtualized GPU, but Metal doesn't work because the userspace compiler plugin is trapped inside the dyld shared cache. You can't extract it. So I reverse-engineered and rewrote it from scratch.
Feb 2026

The Problem

The virtual iPhone uses a paravirtualized GPU (AppleParavirtGPU). The kernel sees it, but Metal doesn't work because the userspace GPU plugin bundle is missing.

On PCC, this bundle is at /System/Library/Extensions/AppleParavirtGPUMetalIOGPUFamily.bundle:

  • AppleParavirtGPUMetalIOGPUFamily — main bundle binary
  • Info.plist
  • _CodeSignature/CodeResources
  • libAppleParavirtCompilerPluginIOGPUFamily.dylib — shader compiler plugin

The main binary, Info.plist, and CodeResources copy straight from the PCC disk image. They work as-is. The compiler plugin dylib is the problem.

Why You Can't Extract the Compiler Plugin from the DSC

libAppleParavirtCompilerPluginIOGPUFamily.dylib does not exist as a standalone file on the PCC disk. It lives only inside the PCC's dyld shared cache (DSC). If you extract it (ipsw dyld extract or similar), the resulting Mach-O has:

  • A vmaddr relative to its position in the shared cache (e.g. 0x19db71000)
  • No LC_DYLD_INFO_ONLY or LC_DYLD_CHAINED_FIXUPS load commands
  • No rebase opcodes, no bind opcodes

dyld can't relocate it or resolve its symbols. backboardd tries to dlopen it, fails, and CoreAnimation crashes in CA::OGL::MetalContext::get_tile_pipeline during shader warmup. backboardd dies, black screen.

Doesn't matter how you extract it — ipsw dyld extract, memory dump, whatever. DSC images only work as part of the shared cache.

What the Compiler Plugin Does

Metal shader compiler plugin for the paravirtual GPU. Loaded via dlopen by CoreAnimation. Four exported C functions:

  • MTLCompilerCreate(targetInfo, version) — Creates a compiler instance. Checks version == 8, stores the target triple, creates a GPU compiler handle via MTLGPUCompilerCreate.
  • MTLCompilerBuildRequestWithOptions(...) — Main compilation function. Takes an LLVM IR module (AIR format), compiles it into a metallib binary, returns the compiled bytes.
  • MTLCompilerDelete(compiler) — Destroys the compiler instance.
  • MTLCompilerReleaseReply(compiler) — Frees the memory buffer from the last compilation.

Internally, MTLCompilerBuildRequestWithOptions:

  • Calls MTLDowngradeAIRModule to downgrade the AIR module for the target
  • Searches LLVM named metadata nodes (air.vertex, air.fragment, air.kernel) for the shader entry point name
  • Creates a shared LLVM module via LLVMExtraMakeSharedModule
  • Builds a metallib using MTLMetalLibCreateExecutableWithTriple
  • Creates a Metal function with MTLMetalFunctionCreate and inserts it
  • Serializes to a memory buffer with MTLWriteMetalLibToMemoryBuffer

External Dependencies

Calls into two libraries already in the iPhone's DSC:

From libGPUCompiler.dylib

  • MTLGPUCompilerCreate / MTLGPUCompilerDestroy
  • MTLDowngradeAIRModule
  • MTLMetalLibCreateExecutableWithTriple
  • MTLMetalFunctionCreate / MTLMetalLibInsertFunction
  • MTLWriteMetalLibToMemoryBuffer

From libLLVM.dylib

  • LLVMDisposeMemoryBuffer / LLVMGetBufferStart / LLVMGetBufferSize
  • LLVMGetTarget / LLVMExtraMakeSharedModule / LLVMExtraDisposeSharedModule
  • LLVMGetNamedMetadataNumOperands / LLVMGetNamedMetadataOperands
  • LLVMGetMDNodeNumOperands / LLVMGetMDNodeOperands / LLVMGetValueName2

All available at runtime — Metal.framework already loads libLLVM and libGPUCompiler.

The Reimplementation

build.sh

#!/bin/zsh
set -e
cd "$(dirname "$0")"

xcrun -sdk iphoneos clang++ \
    -arch arm64e \
    -dynamiclib \
    -std=c++17 \
    -w -Wl,-w \
    -Wl,-undefined,dynamic_lookup \
    -Wl,-not_for_dyld_shared_cache \
    main.mm \
    -o libAppleParavirtCompilerPluginIOGPUFamily.dylib \
    -install_name /System/Library/Extensions/AppleParavirtGPUMetalIOGPUFamily.bundle/libAppleParavirtCompilerPluginIOGPUFamily.dylib \
    -O2

codesign -f -s - libAppleParavirtCompilerPluginIOGPUFamily.dylib

echo "Built and signed $(wc -c < libAppleParavirtCompilerPluginIOGPUFamily.dylib) bytes"

main.mm

// Reimplementation of libAppleParavirtCompilerPluginIOGPUFamily.dylib
// Metal shader compiler plugin for the paravirtualized GPU.
// Bridges between Metal's compiler interface and the GPUCompiler/LLVM pipeline.

#include <cstdlib>
#include <cstring>
#include <cstdint>

// --- LLVM C API (from libLLVM.dylib) ---
extern "C" {
    typedef struct LLVMOpaqueModule *LLVMModuleRef;
    typedef struct LLVMOpaqueMemoryBuffer *LLVMMemoryBufferRef;
    typedef struct LLVMOpaqueValue *LLVMValueRef;

    void LLVMDisposeMemoryBuffer(LLVMMemoryBufferRef MemBuf);
    const char *LLVMGetBufferStart(LLVMMemoryBufferRef MemBuf);
    size_t LLVMGetBufferSize(LLVMMemoryBufferRef MemBuf);
    const char *LLVMGetTarget(LLVMModuleRef M);
    LLVMModuleRef LLVMExtraMakeSharedModule(LLVMModuleRef M);
    void LLVMExtraDisposeSharedModule(LLVMModuleRef M);

    unsigned LLVMGetNamedMetadataNumOperands(LLVMModuleRef M, const char *Name);
    void LLVMGetNamedMetadataOperands(LLVMModuleRef M, const char *Name, LLVMValueRef *Dest);
    unsigned LLVMGetMDNodeNumOperands(LLVMValueRef V);
    void LLVMGetMDNodeOperands(LLVMValueRef V, LLVMValueRef *Dest);
    const char *LLVMGetValueName2(LLVMValueRef Val, size_t *Length);
}

// --- GPUCompiler API (from libGPUCompiler.dylib) ---
extern "C" {
    typedef void *MTLGPUCompilerRef;
    typedef void *MTLMetalLibRef;
    typedef void *MTLMetalFunctionRef;

    MTLGPUCompilerRef MTLGPUCompilerCreate(int version);
    void MTLGPUCompilerDestroy(MTLGPUCompilerRef compiler);
    int MTLDowngradeAIRModule(LLVMModuleRef module, const char *targetTriple, int flags);
    MTLMetalLibRef MTLMetalLibCreateExecutableWithTriple(LLVMModuleRef sharedModule, const char *entryPointData);
    MTLMetalFunctionRef MTLMetalFunctionCreate(LLVMModuleRef sharedModule, const char *entryName);
    void MTLMetalLibInsertFunction(MTLMetalLibRef lib, MTLMetalFunctionRef func);
    LLVMMemoryBufferRef MTLWriteMetalLibToMemoryBuffer(MTLMetalLibRef lib);
}

struct AppleParavirtCompilerTargetInfo {
    const char *targetTriple;
};

class AppleParavirtCompiler {
public:
    virtual ~AppleParavirtCompiler();
    bool init(const AppleParavirtCompilerTargetInfo *info);
    bool buildRequestWithOptions(
        const void *inputData, size_t inputSize, unsigned int flags,
        LLVMModuleRef module,
        const void **outCode, size_t *outCodeSize,
        const void **outReflection, size_t *outReflectionSize,
        const void **outDebug, size_t *outDebugSize,
        const char **outError);
    void deleteCompilerReply();

private:
    LLVMMemoryBufferRef _bufferRef = nullptr;
    MTLGPUCompilerRef _gpuCompiler = nullptr;
    const char *_targetTriple = nullptr;
};

static const char *entryPointsMetadata[] = {
    "air.vertex", "air.fragment", "air.kernel",
};
static const int numEntryPointsMetadata = 3;

AppleParavirtCompiler::~AppleParavirtCompiler() {
    if (_gpuCompiler) {
        MTLGPUCompilerDestroy(_gpuCompiler);
        _gpuCompiler = nullptr;
    }
}

bool AppleParavirtCompiler::init(const AppleParavirtCompilerTargetInfo *info) {
    _targetTriple = info->targetTriple;
    _gpuCompiler = MTLGPUCompilerCreate(1);
    return (_gpuCompiler != nullptr);
}

void AppleParavirtCompiler::deleteCompilerReply() {
    if (_bufferRef) {
        LLVMDisposeMemoryBuffer(_bufferRef);
        _bufferRef = nullptr;
    }
}

bool AppleParavirtCompiler::buildRequestWithOptions(
    const void *inputData, size_t inputSize, unsigned int flags,
    LLVMModuleRef module,
    const void **outCode, size_t *outCodeSize,
    const void **outReflection, size_t *outReflectionSize,
    const void **outDebug, size_t *outDebugSize,
    const char **outError)
{
    _bufferRef = nullptr;
    if (!module) { *outError = "null module"; return true; }

    int result = MTLDowngradeAIRModule(module, _targetTriple, 0);
    if (!result) { *outError = nullptr; return false; }
    if (!_gpuCompiler) { *outError = "no gpu compiler"; return true; }

    const char *entryName = nullptr;
    size_t entryNameLen = 0;

    for (int i = 0; i < numEntryPointsMetadata; i++) {
        unsigned numNamedOps = LLVMGetNamedMetadataNumOperands(module, entryPointsMetadata[i]);
        if (numNamedOps > 0) {
            LLVMValueRef *namedOps = (LLVMValueRef *)alloca(numNamedOps * sizeof(LLVMValueRef));
            LLVMGetNamedMetadataOperands(module, entryPointsMetadata[i], namedOps);
            LLVMValueRef mdNode = namedOps[0];
            if (mdNode) {
                unsigned numOps = LLVMGetMDNodeNumOperands(mdNode);
                if (numOps > 0) {
                    LLVMValueRef *ops = (LLVMValueRef *)alloca(numOps * sizeof(LLVMValueRef));
                    LLVMGetMDNodeOperands(mdNode, ops);
                    if (ops[0]) {
                        entryName = LLVMGetValueName2(ops[0], &entryNameLen);
                        break;
                    }
                }
            }
        }
    }

    const char *triple = LLVMGetTarget(module);
    LLVMModuleRef sharedModule = LLVMExtraMakeSharedModule(module);

    char *entryNameStr = nullptr;
    if (entryName && entryNameLen > 0) {
        entryNameStr = (char *)malloc(entryNameLen + 1);
        memcpy(entryNameStr, entryName, entryNameLen);
        entryNameStr[entryNameLen] = '\0';
    }

    MTLMetalLibRef metalLib = MTLMetalLibCreateExecutableWithTriple(sharedModule,
                                                                     entryNameStr ? entryNameStr : "");
    if (entryNameStr) free(entryNameStr);
    if (!metalLib) {
        LLVMExtraDisposeSharedModule(sharedModule);
        *outError = "failed to create metallib";
        return true;
    }

    MTLMetalFunctionRef func = MTLMetalFunctionCreate(sharedModule, entryName);
    MTLMetalLibInsertFunction(metalLib, func);

    _bufferRef = MTLWriteMetalLibToMemoryBuffer(metalLib);
    if (!_bufferRef) {
        LLVMExtraDisposeSharedModule(sharedModule);
        *outError = "failed to write metallib";
        return true;
    }

    LLVMExtraDisposeSharedModule(sharedModule);
    *outCode = LLVMGetBufferStart(_bufferRef);
    *outCodeSize = LLVMGetBufferSize(_bufferRef);
    *outReflection = nullptr; *outReflectionSize = 0;
    *outDebug = nullptr; *outDebugSize = 0;
    *outError = nullptr;
    return false;
}

// --- Exported C interface ---
extern "C" {

__attribute__((visibility("default")))
void *MTLCompilerCreate(void *targetInfo, uint64_t version) {
    if (version != 8) return nullptr;
    AppleParavirtCompiler *compiler = new AppleParavirtCompiler();
    if (!compiler->init((const AppleParavirtCompilerTargetInfo *)targetInfo)) {
        delete compiler;
        return nullptr;
    }
    return compiler;
}

__attribute__((visibility("default")))
bool MTLCompilerBuildRequestWithOptions(
    void *compilerObj,
    const void *inputData, size_t inputSize, unsigned int flags,
    LLVMModuleRef module,
    const void **outCode, size_t *outCodeSize,
    const void **outReflection, size_t *outReflectionSize,
    const void **outDebug, size_t *outDebugSize,
    const char **outError)
{
    AppleParavirtCompiler *compiler = (AppleParavirtCompiler *)compilerObj;
    return compiler->buildRequestWithOptions(inputData, inputSize, flags, module,
                                             outCode, outCodeSize,
                                             outReflection, outReflectionSize,
                                             outDebug, outDebugSize, outError);
}

__attribute__((visibility("default")))
void MTLCompilerDelete(void *compilerObj) {
    if (compilerObj) delete (AppleParavirtCompiler *)compilerObj;
}

__attribute__((visibility("default")))
void MTLCompilerReleaseReply(void *compilerObj) {
    if (compilerObj) ((AppleParavirtCompiler *)compilerObj)->deleteCompilerReply();
}

} // extern "C"

Key Linker Flags

  • -undefined dynamic_lookup — LLVM and GPUCompiler symbols aren't in the SDK. This defers resolution to runtime, where dyld finds them in the DSC.
  • -not_for_dyld_shared_cache — Required because the linker rejects -undefined dynamic_lookup for shared-cache-eligible dylibs.
  • -install_name ... — Must match the exact on-device load path.

Ad-hoc codesigned. Sufficient on a jailbroken device.

Build & Install

  1. Build the dylib: chmod +x build.sh && ./build.sh
  2. Boot the VM in DFU mode, start the SSH ramdisk: 
    ./.build/debug/tart run vphone --dfu &
cd CFW && ./boot_rd.sh
iproxy 2222 22 &
  3. Mount the rootfs writable and copy the dylib: 
    ssh root@127.0.0.1 -p2222 "/sbin/mount_apfs -o rw /dev/disk1s1 /mnt1"

scp -P 2222 libAppleParavirtCompilerPluginIOGPUFamily.dylib \
    root@127.0.0.1:/mnt1/System/Library/Extensions/AppleParavirtGPUMetalIOGPUFamily.bundle/
  4. Halt and normal boot: 
    ssh root@127.0.0.1 -p2222 "sync && /sbin/halt"
# wait for VM to stop, kill tart
./.build/debug/tart run vphone --vnc-experimental

SSH password is alpine. The rest of the bundle (main binary, Info.plist, CodeResources) should already be on the rootfs from fix_boot.py.

Verify

  • backboardd running (no crash)
  • SpringBoard renders with a background, not a black screen
  • MetalTest prints Metal Device Create Success: Apple Paravirtual device

Reverse Engineering Notes

RE'd from the PCC dyld shared cache. Any disassembler that handles DSC loading works (IDA, Ghidra, Hopper). Open the PCC DSC, navigate to the libAppleParavirtCompilerPluginIOGPUFamily image.

  • 47 total functions, most are C++ thunks/destructors
  • The class is AppleParavirtCompiler with a vtable, three member variables (buffer ref, GPU compiler handle, target triple)
  • Import stubs (auth_stubs in arm64e) confirmed the exact set of LLVM/GPUCompiler functions used
  • The entry point metadata search iterates over air.vertex, air.fragment, air.kernel named metadata nodes, pulls the first operand of the first metadata tuple, and reads the function name
  • The LLVM metadata access in the DSC used C++ LLVM API directly (since both the plugin and libLLVM share the cache address space). The reimplementation uses the older LLVM C API which has been exported since LLVM 3.x
  • MTLDowngradeAIRModule returning 0 means "nothing to compile" (success, no output). Nonzero means compilation should proceed.
  • MTLCompilerCreate checks version == 8. Other versions return null.
